Ce site fut réalisé lors du projet étudiant de la SAÉ 3.03 par Dorian Khireddine et Nino Villoutreix. L'image de la marqué QQF est utilisée à but universitaire et ce site n'est en aucun cas concurrentiel à la marque originale.

Il s'agit d'un site informatif sur la montée des eaux, visant à sensibiliser un public large sur ses dangers, et les solutions possibles.
Nous avons tout d'abord recherché une technique de narration et une DA originale sur figma (https://www.figma.com/design/Kss1OEWH8GmkRncteZlyaU/SAE-303?node-id=0-1&p=f&t=s5jjVIqpiuUxtd5W-0). Le compte rendu est disponible sur notre dossier de synthèse (https://www.canva.com/design/DAG7AcGT8rk/3ox-F2Nx3jP-1IZh46vRSA/edit).

Le projet du site web a été réalisé en tailwind, et utilise GSAP et des animations LottieFiles pour rendre le tout plus intéressant visuellement, et promulguer la DA choisie.
La plupart des animations utilisent GSAP, car les animations Lottie nous étaient partiellement limitées sous peine de payer un abonnement; nous avions également peu de personnalisation possible, et avons dù intégrer la cascade en absolute pour que son espace perdu ne pousse pas les autres éléments.
Enfin, en vue de respecter les normes d'accessibilité, nous proposons un bouton permettant de désactiver les animations (sauf Lottie) à l'arrivée sur le site.

## Key Features

* LivePreview - Make changes, See changes
  - Instantly see what your Markdown documents look like in HTML as you create them.
* Sync Scrolling
  - While you type, LivePreview will automatically scroll to the current location you're editing.
* GitHub Flavored Markdown  
* Syntax highlighting
* [KaTeX](https://khan.github.io/KaTeX/) Support
* Dark/Light mode
* Toolbar for basic Markdown formatting
* Supports multiple cursors
* Save the Markdown preview as PDF
* Emoji support in preview :tada:
* App will keep alive in tray for quick usage
* Full screen mode
  - Write distraction free.
* Cross platform
  - Windows, macOS and Linux ready.

## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com)) installed on your computer. From your command line:

```bash
# Clone this repository
$ git clone https://github.com/amitmerchant1990/electron-markdownify

# Go into the repository
$ cd electron-markdownify

# Install dependencies
$ npm install

# Run the app
$ npm start
```

> **Note**
> If you're using Linux Bash for Windows, [see this guide](https://www.howtogeek.com/261575/how-to-run-graphical-linux-desktop-applications-from-windows-10s-bash-shell/) or use `node` from the command prompt.
